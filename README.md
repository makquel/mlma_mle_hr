## Commands

- run:

```bash
source venv/bin/activate; cd src; python3 app.py
```

- install:

```bash
bash install.sh; source venv/bin/activate; pip3 install -r requirements.txt
```

- test:

```bash
source venv/bin/activate; cd src; py.test -p no:warnings
```
